OCF Policies are established by the [OCF Decision Making Process](/user-docs/operatingrules/constitution/#ocf-decision-making-process).

These are the current policies of the Open Computing Facility:

 * [Account policy](/user-docs/services/account/account-policy/)
 * [Eligibility policy](/user-docs/membership/eligibility/)
 * [Banning policy](/user-docs/membership/banning/)
 * [Content removal policy](/user-docs/services/account/content-removal/)
 * [Lab reservation policy](/user-docs/services/lab/lab-reservation-policy/)
 * The printing quotas as documented on the
   [Printing](/user-docs/services/lab/printing/) page
 * [Database policy](/user-docs/services/mysql/#policies)
 * [Disk quota policy](/user-docs/services/shell/#disk-quotas)
 * [Unattended processes policy](/user-docs/services/shell/#unattended-processes)
 * [Virtual hosting policy](/services/vhost/#policies)

There are also OCF policies which apply only to staff. These policies can
be found at the [Staff policies](/staff-docs/policies/) page.

University policies additionally apply to the OCF. A list of the University's
IT policies can be found [here][university-policy]. Some policies of note
include:

 * [Computer Use Policy](https://security.berkeley.edu/computer-use-policy)
 * [Campus Online Activities Policy](https://security.berkeley.edu/campus-online-activities-policy)
 * [Electronic Communications Policy](https://security.berkeley.edu/electronic-communications-policy)

[university-policy]: https://security.berkeley.edu/policy/policy-catalog
